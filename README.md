# AI Voice Assistant

## Project Overview
This is a Python-based AI Voice Assistant that performs voice-controlled tasks using speech recognition and text-to-speech technology.

## Features
- Voice Recognition
- Text-to-Speech
- Google Search
- YouTube Search
- Wikipedia Search
- Open Applications
- To-Do List Management
- Desktop Notifications
- WhatsApp Message Scheduling
- Email Sending
- ChatGPT Opening

## Technologies Used
- Python
- pyttsx3
- SpeechRecognition
- Wikipedia
- PyAutoGUI
- Plyer
- smtplib
